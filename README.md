## **📑 Info**

This is a mod that extends [AE2][ae2]. It adds new mechanics and items that make working with the ME system more convenient.

## **✨ Added**

* New storage cells and crafting storage: 1M, 4M, 16M, 64M, and 256M
* An advanced version of the acceleration card
* A stack card used by the mechanisms to produce by the stack
* Advanced version of the pattern provider
